ft_printf

説明

C言語標準ライブラリにあるprintfを変換指定子と返り値とフォーマット文字列のフォーマットに絞り実装しました。

苦労した点

メモリリークのチェックとその修正、%pや%xの文字を16進数の整数に変換するアルゴリズムの実装、
writeによるフォーマット文字の失敗時の処理がとてもむずかしかったです。
freeをこの課題から使うことが多くなりメモリリークのチェックを小規模の関数単位ではなくプログラム単位で意識する必要があり、
メモリリーク探索にはlinuxではvalgrind MacOSではfsanitizeを使ってメモリリークの箇所を検知しデバックを行っていました。
